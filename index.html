Intelligent Transport System

Tiny YOLO Lego Detection and Counting

This intelligent transport system project aims to reduce manpower needed and coordination work for an autonomous vehicle test trial by building dynamic roadblocks. Cameras are installed to monitor traffic conditions so that the traffic light can make a better decision on what the logic should be. To determine the traffic situation, tiny-YOLO, a computer vision model is used to detect and count the vehicle.
